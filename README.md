Это чекер цен на авиабилеты компании WizzAir. В файлике routes.json выставляем откуда, куда и когда хотите лететь. Структура этого файла такова:
[
  {
    "id": "rmo-otp-2025-12-25",  <=========== Это можно не трогать, это не учавствует в поиске
    "origin": "OTP",             <=========== Откуда вылетаем. Код аэропорта можно найти в airports.json
    "destination": "TFS",        <=========== Куда хотим лететь, код берем там же
    "depart_date": "2026-07-08", <=========== Дата вылета, год-месяц-число (желательно посмотреть на сайте когда есть рейсы, а потом эту дату сюда и вписать)
    "return_date": null,         <=========== Дата обратного вылета, если нужна. Если не нужна - оставляем так 
    "adults": 1,                 <=========== Количество взрослых 
    "cabin_class": "economy"     <=========== Класс билета
  }, <=========== Если не нужно большеничего искать, можно удалить, но надо удалять все, начиная с этой запятой и до ]
  {
    "origin": "TFS", <=========== тут все по аналогии как выше
    "destination": "OTP",
    "depart_date": "2026-07-08",
    "return_date": null,
    "adults": 1,
    "cabin_class": "economy"
  },
  {
    "origin": "RMO",
    "destination": "OTP",
    "depart_date": "2026-07-08",
    "return_date": null,
    "adults": 1,
    "cabin_class": "economy"
  }
]

В Botfather регистрируем нового бота, берем его TELEGRAM_TOKEN и прописываем в .env. Там же нужно прописать CHAT_ID можно узнать через getUpdates

Бота можно деплоить куда-то на VPS, или запустить на компе локально(комп, что не выключается, чтоб мониторинг был постоянным)
